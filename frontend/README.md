# Frontend — Plataforma Imobiliária

Painel web (admin + app imobiliário). Todo código aqui faz parte do **Projeto-X**.

## Responsabilidade

- Login e layout base
- Dashboard
- CRM: Imóveis, Contatos, Pipeline (Kanban), Tarefas
- Configuração do agente (nome, prompt) e conexão WhatsApp (QR)
- (Fases seguintes) Disparos, listas/tags, config do site, Instagram

## Stack prevista

- React (ou Next.js), TypeScript, Vite
- Chamadas à API do backend

## Desenho

Ver `../docs/DESENHO_FASE_0.md`.

## Como iniciar desenvolvimento

1. Criar projeto (ex.: `pnpm create vite` ou Next.js).
2. Configurar roteamento e layout; tela de login.
3. Implementar páginas do CRM (lista de imóveis, contatos, pipeline).
